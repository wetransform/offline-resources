OGC(r) CoverageJSON (CovJSON)
-----------------------------------------------------------------------

The OGC CoverageJSON Community Standard is a format for publishing
spatiotemporal data to the Web and is based on JavaScript Object
Notation (JSON). The primary design goals are simplicity, machine and
human readability and efficiency. While other use cases are possible,
the primary CoverageJSON use case is enabling the development of
interactive visual websites that display and manipulate environmental
data within a web browser.

More information may be found at
 https://www.ogc.org/standards/covjson/

The latest schemas are available for download individually or as a
bundle at http://schemas.opengis.net/covjson/ .
Additionally, you may download all OGC schema in a comprehensive package
from http://schemas.opengis.net/SCHEMAS_OPENGIS_NET.zip .

* Latest version is: http://schemas.opengis.net/covjson/ *
